- Real-time timer (MM:SS format)

- Time category selection
- Question theme/subject selection
- Notes field for observations
- Start/Pause/Stop/Reset controls
- Live statistics display

- Automatic file creation

- Session header with description
- Timestamped records (ISO 8601)
