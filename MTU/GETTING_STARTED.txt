## ğŸ› Troubleshooting

### Problem: Application won't launch
```
Solution:
1. Check Python version: python --version
2. Try: python -m tkinter (should open test window)
3. If error, reinstall Python 3.8+
```

### Problem: tkinter test fails
```
Solution:
Windows: Usually included with Python
macOS: May need: brew install python-tk
Linux: apt-get install python3-tk
```

### Problem: Can't find CSV file
```
Solution:
1. Check folder: mock_test_logs/
2. If not there, create folder manually
3. Restart app and try again
```

### Problem: Timer not updating
```
Solution:
1. Restart application
2. Try simple test first (1 question)
3. Check system resources
```
